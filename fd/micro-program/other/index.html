                          # 实际项目中的问题

                          # 同一个链接，先后顺序跳转至不同页面

                          本地保存用户的pv

                          # 小视频随机播放

                          使用到 video的 autoplay，播放结束事件 bindended 和播放失败 binderror

                          1. 获取设备信息
                          const SYS = wx.getSystemInfoSync()
                          1. 获取元素信息，更新随机队列
                          wx.createSelectorQuery().select('#id').fields({
	size: true,
	rect: true
}, function (res) {
	// res { width, height, top, bottom, left, right }
}).exec()
                          1. 页面滚动触发随机，更新随机队列
                          onPageScroll(e) {
	randomPlay(e.scrollTop)
}

                          提示

                          iphone 手机播放失败事件会有问题

                          # 视频圆角

                          原生组件不支持border-radius属性，使用图片遮罩模拟效果

                          # 手机虚拟栏高度问题

                          使用相对单位 vw, vh

                          # 全屏设置

                          修改app.json文件

                          {
	"window": {
		"navigationStyle": "custom"
	}
}

                          全屏注意点

                          1. 返回按钮消失，需要自己实现

                          2. 自己实现的效果在不同手机尺寸上位置会和右侧的 ··· 不一致

                          3. 使用 getCurrentPages 控制路由

                          # cover-view 的对齐问题

                          多嵌套一层 cover-view

                          # 音频视频，页面隐藏时，未暂停

                          手动暂停和恢复

                          # 全屏上下滑手势

                          1. 事件模拟 下拉可以使用 onPullDownRefresh 事件，操作完成后记得调用 wx.stopPullDownRefresh()

                          针对android端:

                          上拉使用 onReachBottom 模拟

                          ios： 上拉使用 onPageScroll 模拟

                          恢复滚动状态： wx.pageScrollTo({ scrollTop: 0 })

                          1. 组件模拟

                          swiper 组件可实现上下滑动，体验太差

                          # cookie或者session无法写入

                          使用本地存储替换，手动注入sessionid或者cookie

                          # 画布文字渐变

                          画布上的文字渐变要以每一行为一个渐变单位

                          # 画布文字加粗

                          文字加粗效果，以重复几次绘制即可

                          # tab页

                          使用 onTabItemTap 添加tab 双击功能
