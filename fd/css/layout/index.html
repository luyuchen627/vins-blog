        # 百分比

        使用相对单位可以很好的完成自适应的页面

        • em: 相对父级字体大小；设置于border和padding，可以保持留白的比例性

        • rem: 相对于根元素的字体大小；整体相对于根字体大小来适配页面；在字体图标上也有一番作为

        • %：相对父容器的比例

        • vw/vh: 相对于屏幕的相对单位

        # flex

        # 定义

        语法

        dispaly: flex | inline-flex;

        WARNING

        设置为felx后，子元素的float，clear和vertical-align属性将失效

        # 容器的属性

        1. 主轴方向

        flex-direction

        flex-direction：row | row-reverse | column | column-reverse;

        参数说明：

        • row(默认值)：水平方向，起点在左端
        • row-reverse：水平方向，起点在右端
        • column：垂直方向，起点在顶部
        • column-reverse：垂直方向，起点在底部
        1. 换行

        flex-wrap

        flex-wrap：nowrap | wrap | wrap-reverse;

        参数说明：

        • nowrap(默认值)：不换行
        • wrap：正常换行
        • wrap-reverse：反向换行
        1. flex-flow

        lex-flowf 为 flex-direction 和 flex-wrap 的简写

        flex-flow

        flex-flow：direction wrap;

        1. 主轴的对齐方式

        主轴的对齐方式

        justify-content: flex-start | flex-end | center | space-between | space-around;

        1. 侧轴的对齐方式

        侧轴的对齐方式

        align-items: flex-start | flex-end | center | baseline | stretch;

        1. 多行对齐方式

        多行对齐方式

        align-content: flex-start | flex-end | center | space-between | space-around | stretch;

        # 容器项目的属性

        1. 排列顺序

        排列顺序

        order: 数字;

        数值越小，排列越靠前，默认为0

        1. 项目扩展

        项目扩展

        flex-grow: 数字;

        存在剩余空间时的扩展比例，默认为0

        1. 项目缩小

        项目缩小

        flex-shrink: 数字;

        存在剩余空间时的缩小比例，默认为1

        1. 项目预占位值

        项目预占位值

        flex-basis: 长度;

        存在剩余空间时, 未扩展和缩小之前项目的预先占用的宽度，默认为auto

        1. flex

        flex 为 flex-grow, flex-shrink 和 flex-basis 的缩写

        flex-flow

        flex-：grow shrink basis;

        1. 项目对齐

        项目对齐

        align-self: auto | flex-start | flex-end | center | baseline | stretch;

        # 应用场景

        • 自适应布局

        • 等高布局

        • 居中

        # grid

        # 定义

        语法

        dispaly: grid | inline-grid;

        WARNING

        设置为grid后，子元素的float，clear和vertical-align属性将失效

        # 设置网格的行和列

        语法

        // 设置网格的行 grid-template-rows: row1 row2 ... rown;

        // 设置网格的列 grid-template-columns: row1 row2 ... rown;

        参数说明: row1 表示行高或者列宽，单位可以是 px，%，em 等任何非负值，以及特殊单位 fr 和 minmax 函数，也可以使用 repeat 函数创建重复的网格

        fr 单位：网格的可用空间的等分比例

        minmax 函数：接受两个参数；第一个参数为最小值，第二个参数为最大值，参数可以是任何长度值，以及 auto

        repeat 函数：接受两个参数；第一个参数为重复的次数，第二个参数为定义每个轨道的尺寸

        设置 grid-template-rows 和 grid-template-columns 后，网格会被分为 n*m 个网格项目，再不改变顺序的情况下，容器中的项目会依序填充网格，多余的项目保持自己的宽高

        # 剩余项目的设置(隐式网格)

        语法

        grid-auto-rows：10px; //行宽

        grid-auto-column：10px; // 列高

        grid-auto-flow：row | column; // 网格方向

        # 设置网格之间的间距

        语法

        // 设置网格的行间距 grid-rows-gap: gap;

        // 设置网格的列间距 grid-columns-gap: gap;

        // 设置网格的行列间距 grid-gap: row-gap columns-gap;

        参数说明: gap 表示间距，单位可以是 px，%，em 等任何非负值

        参考文档

        # 容器项目的重新定位

        默认情况下，一个网格项目只会占用容器中的一个网格，可以使用 grid-row，grid-column，grid-area 通过网格线来对网格项目分配网格

        网格线默认以 1 开始

        语法

        grid-row-start: 2; grid-row-end: 3; grid-column-start: 2; grid-column-end: 3;

        grid-row grid-row 语法

        grid-row: start / end;

        grid-row: start / span 1;

        使用 span 表示跨越的行(列)数

        grid-area语法

        grid-area: row-start / column-start / row-end / column-end;

        grid-area: 2 / 2 / 3 / 3;

        当两个网格项目重叠的时候，可以使用 z-index 来调整层叠顺序

        # 网格项目对齐方式

        网格项目对齐方式和flex容器的对齐几乎形同
