        # css黑科技

        # 属性

        # 方法

        • attr() 配合content，获取元素属性
        • calc() 计算
        • var() 自定义变量

        # 保留值

        • inherit 继承
        • initial 默认值
        • currentColor 指向当前的color
        • max-content
        • min-content 这个关键字将解析为这个容器内部最大的不可断行元素的宽度（即最宽的单词、图片或具有固定宽度的盒元素）
        • fit-content
        • ch单位解决等宽字体

        # css属性

        • pointer-events: 鼠标事件
        • text-align-last: justify; 两端对齐，不止文字（text-align:justify不对快内的最后一行文本做处理）
        • object-fit
        • mix-blend-mode。将其值设置为difference（差值模式）或者darken（变暗模式）
        • resize 调整元素的大小（overflow为visible会无效），可以使用伪元素优化拖动标识
        • margin：0 calc(50% - 100px); 实现元素固定宽度，同时居中的效果，padding也可以

        # 选择器

        • :valid 和 :invalid 来做表单即时校验
        • :not() 去除不想包含的匹配
        • :only-child
        • 使用:first-child:nth-last-child(n) 来实现:only-nth-child(n)的效果，:only-nth-child(n)，意为有且只有n个元素的匹配，目前不存在这个选择器，相应的如果把n写为n+a，则为至少有a个元素的匹配，类似的把n写为n-b，则为至多有b个元素的匹配

        # 边框

        • 模拟图形
        • 替换margin，配合怪异盒模型，应用百分比布局场景

        # 阴影系列

        # 文字模糊

        { 
    color: transparent;
    text-shadow: #111 0 0 5px;
}

        通过text-shadow还可以实现发光，镂空，3d等效果

        # 多重边框

        {
	background: #fff;
	box-shadow: 0 0 0 10px #655,
		0 0 0 15px deeppink,
		0 2px 5px 15px rgba(0,0,0,.6);
}

        注意：box-shadow 是层层叠加的，第一层投影位于最顶，而且不影响布局

        # 边框内圆角

        {
	background: #fff;
	border-radius: .8em;
	padding: 1em;
	box-shadow: 0 0 0 .6em #655;
	outline: .6em solid #655;
}

        注意： box-shadow的扩展不能小于 border-radius的值的一半(√2-1)

        # 多重边框

        {
    box-shadow: 0 0 0 6px rgba(0, 0, 0, 0.2), 0 0 0 12px rgba(0, 0, 0, 0.2), 0 0 0 18px rgba(0, 0, 0, 0.2), 0 0 0 24px rgba(0, 0, 0, 0.2);
    height: 100px;
    width: 100px;
}

        使用filter: drop-shadow代替box-radius

        # 条纹背景

        {
	background: linear-gradient(#fff 0, #ff0 33.3%, #f00 66.6%);
	background-size: 100% 30px;
}

        注意：条纹的方向使用linear-gradient的角度可以调节，宽度或者高度使用background-size调节

        拓展: 可以使用多重渐变绘制图形
