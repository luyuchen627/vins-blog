            # css中的自定义变量

            # 定义

            由于'$'和'@'分别被sass和less占用，原生css使用 -- 来定义变量

            body {
  --default-color: #123456;
  --default-size: 24;
  --default-bg-color: var(--default-color);
}

            注意：

            • 变量名区分大小写
            • 变量值可以是css中的各种合法类型
            • 变量值也可以是了一个变量

            # 使用

            使用 var() 来使用变量

            语法

            var( --var, [default])

            参数说明： --var 为变量，如果 --var 不存在，则使用 default 作为默认值

            body {
  --default-color: #123456;
  --default-size: 24;
  color: var(--default-color);
  font-size: calc(var(--default-size) * 1px);
}

            变量的类型运算注意：

            • 字符串：自接拼接，不需要使用'+'
            • 数字：需要使用calc运算转化

            # 作用域

            变量的作用域和css的选择器作用域一样

            <style>
  :root { --color: blue; }
  div { --color: green; }
  #box { --color: red; }
  * { color: var(--color); }
</style>

<h2>blue</h2>
<div>green</div>
<div id="box">red</div>

            # js操作css变量

            • 设置变量 element.style.setProperty('--color', 'blue')

            • 读取变量 element.style.getPropertyValue('--color')

            • 读取真实渲染的样式 getComputedStyle(element).color

            • 删除变量 element.style.removeProperty('--color')

            • 检测兼容性 使用window.CSS.supports函数，返回一个Boolean CSS.supports("text-decoration-style", "blink") CSS.supports("display: flex") CSS.supports("(--foo: red)")

            变量的读取只是读取设置的值，并不是实际渲染的样式，如果读取实际渲染的样式，请使用 getComputedStyle 函数

            # 场景

            • 切换主题
            • 适用于定义一套样式风格
            • 用js修改样式
