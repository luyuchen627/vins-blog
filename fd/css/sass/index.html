                # sass 基础

                # sass语法格式

                Sass的两种语法格式：

                1. SCSS：在css3的基础上的扩展，文本以.scss结尾
                2. SASS：css的简化模式，它使用 “缩进” 代替 “花括号” 表示属性属于某个选择器，用 “换行” 代替 “分号” 分隔属性，文本以.sass结尾

                # 常用语法(以scss为例)

                # 嵌套

                1. 嵌套规则

                嵌套可以作用于选择器，可以作用于属性

                .a {
	.b {
		color: red;
		font: {
			family: fantasy;
			size: 30em;
			weight: bold;
		}
	}
}

// 编译结果
.a .b { 
	color: red; 
	font-family: fantasy;
	font-size: 30em;
	font-weight: bold;
}
                1. 父选择器

                使用 & 来选择父元素

                .a {
	.&_b {
		color: red;
	}
}

// 编译结果
.a .a_b { color: red; }
                1. 跳出嵌套

                使用 @at-root跳出嵌套





                 








                 



                .nav {
	#{&}-lt {
		color: red;
	}
	@at-root #{&}-rt {
		color: blue;
	}
}
 
// 编译结果
.nav .nav-lt {
  color: red;
}
.nav-rt {
  color: blue;
}

                # 注释

                1. /* */: 编译后会写入css文件

                2. //: 编译后不会写入css文件

                # 变量

                变量使用 $ 来定义，和js变量一样，先定义，在使用；同时变量也有作用域的限制

                1. 默认变量，在定义的变量后面加上 !default

                2. 在块级作用域定义全局变量，在定义的变量后面加上 !global

                # 变量类型

                1. 数字，1, 2, 13, 10px

                2. 字符串，有引号字符串与无引号字符串，"foo", 'bar', baz

                3. 颜色，blue, #04a3f9, rgba(255,0,0,0.5)

                4. 布尔型，true, false

                5. 空值，null

                6. 数组 (list)，用空格或逗号作分隔符，1.5em 1em 0 2em, Helvetica, Arial, sans-serif

                7. maps, 相当于 JavaScript 的 object，(key1: value1, key2: value2)

                list类型和map类型的取值，参考 函数的使用

                # 运算

                1. 数字运算：支持数字的加减乘除、取整等运算 (+, -, *, /, %)

                2. 关系运算：<, >, <=, >= 也可用于数字运算，相等运算 ==, != 可用于所有数据类型

                3. 插值语句: #{}, #{变量}会返回无符号字符串

                4. 布尔运算：支持布尔型的 and or 以及 not 运算

                5. 字符运算: + 可用于连接字符串 有引号字符串（位于 + 左侧）连接无引号字符串，运算结果是有引号的，相反，无引号字符串（位于 + 左侧）连接有引号字符串，运算结果则没有引号

                6. 数组和list操作，见相关的内置函数

                # 导入@import

                1. @import导入sass文件，可以不用扩展名，导入css文件和css中的@import无异

                2. sass文件名前添加下划线，这样会告诉 Sass 不要编译这些文件

                3. @import在嵌套层导入，导入的样式只能出现在嵌套的层中

                # 继承@extend

                使用 @extend 语句来继承选择器(例如：类，id或者标签)的样式，也可以继承以 % 开头的占位符的样式

                .error {
	color: red;
	font-weight: bold;
}
.error-tip {
	@extend .error;
	font-size: 14px;
}

// 编译结果
.error, .error-tip {
	color: red;
	font-weight: bold;
}
.error-tip {
	font-size: 14px;
}

                # 混入@mixin

                Sass中使用@mixin声明混入，可以传递参数，参数名以$符号开始，多个参数以逗号分开，也可以给参数设置默认值

                声明的@mixin通过@include来调用

                // 添加浏览器前缀
@mixin prefix($prop, $val) {
	@each $prefix in -webkit-, -moz-, -ms-, -o-, '' {
		#{$prefix}#{$prop}: $val;
	} 
}
@mixin border-radius($val) {
	@include prefix(border-radius, $val);
}

// 例子
h2 {
	@include border-radius(4px);
}

// 编译结果
h2 {
	-webkit-border-radius: 4px;
	-moz-border-radius: 4px;
	-ms-border-radius: 4px;
	-o-border-radius: 4px;
	border-radius: 4px;
}

                剩余参数

                sass也可以像js那样使用剩余参数，只是扩展符在变量后面

                @mixin prefix(#var...) { ... }

                # 内容占位@content

                // 定义mixin
@mixin max-screen($res) {
  @media only screen and (max-width: $res) {
    @content;
  }
}

// 例子
@include max-screen(480px) {
  body {
    color: red;
  }
}
 
// 编译结果
@media only screen and (max-width: 480px) {
  body {
    color: red;
  }
}

                # 指令

                1. @if 判断

                语法

                @if [ @else if ] [ @else ]

                1. @for 循环

                语法

                @for $var from start (through || to) end

                参数说明：through 包含 end 的值，to 不包含, start 和 end 必须是整数值

                1. @each 遍历list类型

                语法

                @each $var in list

                参数说明：$var 可以像es6中使用模板匹配

                1. @while 循环

                语法

                @while $var > 0

                1. 混合指令

                定义使用 @mixin， 引用使用 @include，@mixin可以简写为 =，@include可以简写为 +，@mixin的参数和函数定义的参数一样

                1. @content 内容占位

                2. @function 函数

                # 常用内置函数

                # 颜色函数

                1. rgb($red, $green, $blue): 从红色, 绿色, 蓝色和中创建一个颜色

                2. rgba($red, $green, $blue, $alpha)

                3. red($color): 获取颜色的红色分量

                4. green($color): 获取颜色的绿色成分

                5. blue($color): 获取颜色的蓝色分量

                6. mix($color1, $color2, [$weight]): 将两种颜色混合在一起

                7. hsl($hue, $saturation, $lightness): 从色调, 饱和度和渐变中创建一个颜色

                8. hsla($hue, $saturation, $lightness, $alpha)

                9. hue($color): 获取颜色的hue组件

                10. saturation($color): 获取颜色的饱和度分量

                11. lightness($color): 获取颜色的亮度分量

                12. adjust-hue($color, $degrees): 改变颜色的色调

                13. lighten($color, $amount): 使颜色变浅

                14. darken($color, $amount): 使颜色变深

                15. saturate($color, $amount): 使颜色更饱和

                16. desaturate($color, $amount): 使颜色不饱和

                17. grayscale($color): 将颜色转换为灰度

                18. complement($color): 返回颜色的补码

                19. invert($color, [$weight]): 返回颜色的倒数

                # 字符串函数

                1. unquote($string): 从字符串中删除引号
                2. quote($string): 在字符串中添加引号
                3. str-length($string): 返回字符串中的字符数
                4. str-insert($string, $insert, $index): 在$index中将$insert插入$string
                5. str-index($string, $substring): 返回$string中第一次出现$substring的索引
                6. str-slice($string, $start-at, [$end-at]): 从$string中提取子字符串
                7. to-upper-case($string): 将字符串转换为大写
                8. to-lower-case($string): 将字符串转换为小写

                # 数字函数

                1. percentage($number): 将无单位数转换为百分比
                2. round($number): 将数字舍入到最接近的整数
                3. ceil($number): 将一个数字舍入到下一个整数
                4. floor($number): 将数字减少到前一个整数
                5. abs($number): 返回数字的绝对值
                6. min($numbers ...): 找到几个数字的最小值
                7. max($numbers ...): 查找最多几个数字
                8. random(): 返回一个随机数

                # 列表函数

                Sass中的list是不可变的; 所有列表函数都返回一个新列表, 而不是就地更新现有列表 所有list函数也适用于map, 将它们视为对列表

                1. length($list): 返回列表的长度
                2. nth($list, $n): 返回列表中的特定项
                3. set-nth($list, $n, $value): 替换列表中的第n个项目
                4. join($list1, $list2, [$separator, $bracketed]): 将两个列表合并为一个
                5. append($list1, $val, [$separator]): 将单个值追加到列表的末尾
                6. zip($lists ...): 将多个列表组合到一个多维列表中
                7. index($list, $value): 返回列表中值的位置
                8. list-separator($list): 返回列表的分隔符
                9. is-bracketed($list): 返回列表是否包含方括号

                # 数组函数

                map和list一样，也是不变的; 所有map函数都返回一个新map, 而不是就地更新现有map

                1. map-get($map, $key): 返回与给定键关联的映射中的值
                2. map-merge($map1, $map2): 将两个map合并为一个新map
                3. map-remove($map, $keys ...): 返回一个删除了键的新map
                4. map-keys($map): 返回map中所有键的列表
                5. map-values($map): 返回map中所有值的列表
                6. map-has-key($map, $key): 返回map是否具有与给定键关联的值
                7. keywords($args): 返回传递给带变量参数的函数的关键字

                # 内置函数

                1. feature-exists($feature): 返回当前Sass运行时中是否存在某个功能

                2. variable-exists($name): 返回当前作用域中是否存在具有给定名称的变量

                3. global-variable-exists($name): 返回全局范围内是否存在具有给定名称的变量

                4. function-exists($name): 返回是否存在具有给定名称的函数

                5. mixin-exists($name): 返回是否存在具有给定名称的mixin

                6. content-exists(): 返回当前mixin是否传递了内容块

                7. inspect($value): 返回值在Sass中表示的字符串表示形式

                8. type-of($value): 返回值的类型

                9. unit($number): 返回与数字关联的单位

                10. unitless($number): 返回一个数字是否有单位

                11. compare($number1, $number2): 返回是否可以添加, 减去或比较两个数字

                12. call($function, $args ...): 动态调用get-function返回的Sass函数引用

                13. get-function($name, $css: false): 在当前词法范围内查找具有给定名称的函数
