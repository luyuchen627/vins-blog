            # dom操作

            # 节点分类

            • 元素节点(1) element
            • 属性节点(2) attribute
            • 文本节点(3) text
            • 注释节点(8)
            • 文档节点(9) document

            # NodeList 和 HTMLCollection

            • NodeList 和 HTMLCollection 都是集合，类数组对象

              集合表示根集于根集合的子树的视图，仅包含与给定过滤器匹配的节点。视图是线性的，在没有特定要求的情况下，集合中的节点必须以树的顺序排序 集合可以是实时的，也可以是静态的，没有特殊说明，默认都是动态的

            • NodeList 对象是节点的集合，HTMLCollection是元素的集合

              NodeList 最初设计是给 XML 使用的，但是在 DOM 中就会所有相匹配的节点，比如 el.childNodes 会返回所有的子节点，也会包括注释节点，这显然不是我们想要的； 而 HTMLCollection 只返回元素节点，例如 el.children 和 getElements* 系列

            • 内部方法和属性

              NodeList拥有以下方法：

              • item()
              • keys()
              • values()
              • entries()
              • foreach()
              • length

              HTMLCollection只拥有item()方法和length属性

            # 节点操作

            1. 创建新节点
            • document.createElement(tagName): 创建一个具体的元素
            • document.createTextNode(text): 创建一个文本节点
            • node.cloneNode(true/false): 克隆一个节点,它接收一个bool参数，用来表示是否复制子元素
            • document.createDocumentFragment(): 创建一个DOM片段, 用来存储临时节点，文档片段存在于内存中，并不在DOM树中,因此大量操作DOM时用它可以大大提升性能
            1. 添加、移除、替换、插入
            • parent.appendChild(child): 添加子节点，如果child已经存在，则删除，但会保留事件
            • parent.removeChild(child): 删除子节点
            • parent.replaceChild(newNode, oldNode): 替换节点
            • parent.insertBefore(newNode, refNode): 在refNode前插入一个newNode节点，如果refNode是undefined或null，则insertBefore会将节点添加到末尾；
            1. 查找节点

            • el.getElementById()：根据节点ID查找元素，大小写敏感，如果有多个结果，只返回第一个

            • el.getElementsByClassName()：根据类名查找元素，多个类名用空格分隔，返回一个 HTMLCollection

            • el.getElementsByTagName()：根据标签查找元素，* 表示查询所有标签，返回一个 HTMLCollection

            • el.getElementsByName()：根据元素的name属性查找，返回一个 NodeList

            • el.querySelector()：返回单个 Node 如果匹配到多个结果，只返回第一个

            • el.querySelectorAll()：返回一个静态的 NodeList

            • el.childNodes: 子节点，返回一个NodeList

            • el.children: 子元素，返回一个HTMLCollection

            • el.parentNode: 父节点(可能是element,document或者DocumentFragment)

            • el.parentElement: 父元素(element),没有则为null

            • el.childElementCount：返回子元素（不包括文本节点和注释）的个数

            • el.firstChild | el.firstElementChild：指向第一个子元素

            • el.lastChild | el.lastElementChild：指向最后一个子元素

            • el.previousSibling | el.previousElementSibling：指向前一个同辈元素

            • el.nextSibling | el.nextElementSibling：指向最后一个同辈元素

            含有Element的属性只返回元素节点，会忽略到文本节点和注释节点

            # 属性的修改

            • setAttribute(name, value): 设置属性
            • getAttribute(name): 获取属性
